# Lightframe
A simplest and the best web framework. Using this framework, you can create a cool website.
Made with :heart: by <a href="http://github.com/schedule-dash">Schedule Dash development team</a>

# How i can conennect it with my website?
You must unpack it to any folder on your website file storage.
In pages connect it with "link" attribute:<br>
<code>&lt;link rel="stylesheet" href="style.css" type="text/css"/&gt;</code>
